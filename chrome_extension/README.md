# HanyangAuto Extension

한양대학교 LMS에서 동영상 강의 진행을 보조하는 Chrome 확장 프로그램입니다.

이 확장 프로그램은 별도 서버 없이 현재 로그인된 브라우저 세션을 그대로 사용합니다. 사용자가 `learning.hanyang.ac.kr`에 직접 로그인한 뒤, 확장 프로그램이 강의 목록 탐색, 강의 페이지 이동, 플레이어 재생 시도, 학습 상태 확인을 반복합니다.

## 주요 기능

- LMS 대시보드에서 수강 중인 과목과 강의 항목을 수집
- 동영상 강의 항목을 순서대로 열고 자동 재생 시도
- `학습 상태 확인` 버튼을 주기적으로 눌러 완료 여부 재확인
- 이미 처리한 강의 URL을 로컬 저장소에 기록해 중복 처리 방지
- 필요할 때 페이지 내 디버그 패널 표시

## 준비 사항

- Chrome 브라우저
- 한양대학교 LMS 계정
- `https://learning.hanyang.ac.kr` 로그인 상태

## 설치 방법

1. 이 폴더를 내려받거나 공개 레포를 clone 합니다.
2. Chrome에서 `chrome://extensions`로 이동합니다.
3. 우측 상단의 `개발자 모드`를 켭니다.
4. `압축해제된 확장 프로그램 로드`를 클릭합니다.
5. 이 `chrome_extension` 폴더를 선택합니다.

## 사용 방법

1. Chrome에서 `https://learning.hanyang.ac.kr`에 로그인합니다.
2. LMS 탭을 활성화한 상태로 확장 프로그램 팝업을 엽니다.
3. 필요하면 `기존 완료 기록 초기화 후 시작` 옵션을 켭니다.
4. `자동 수강 시작` 버튼을 누릅니다.
5. 진행 중 상태와 디버그 정보를 팝업에서 확인합니다.
6. 중단이 필요하면 `중지` 버튼을 누릅니다.

## 옵션 설명

- `기존 완료 기록 초기화 후 시작`
  - 이전 실행에서 저장된 완료 강의 기록을 비우고 다시 시작합니다.
- `페이지 디버그 패널 표시`
  - 현재 LMS 페이지 위에 상태 패널을 띄워 단계별 진행 상황을 표시합니다.

## 권한 설명

- `storage`
  - 진행 상태, 강의 대기열, 완료 강의 기록, 디버그 설정을 저장합니다.
- `activeTab`
  - 현재 활성 LMS 탭을 기준으로 자동화를 시작하고 제어합니다.
- `tabs`
  - 강의 페이지로 탭을 이동시키고 현재 탭 상태를 갱신합니다.
- `scripting`
  - LMS 페이지 문맥에서 자동화 로직을 실행합니다.
- `debugger`
  - 일부 HYCMS 플레이어 시작 버튼이 일반 클릭으로 반응하지 않을 때 실제 좌표 클릭을 보내는 데 사용합니다.
- `https://*.hanyang.ac.kr/*`
  - LMS 및 관련 학습 플레이어 페이지에서만 동작합니다.

## 동작 방식

1. 대시보드 API에서 수강 과목 목록을 읽습니다.
2. 각 과목의 모듈 항목 중 동영상 출석 항목을 찾습니다.
3. 강의 페이지로 이동합니다.
4. 출석 iframe과 HYCMS 플레이어 iframe 안에서 재생 가능 상태를 확인합니다.
5. 재생 또는 이어보기를 시도합니다.
6. `학습 상태 확인`을 반복해 완료 상태가 반영되면 다음 강의로 이동합니다.

## 제한 사항

- LMS 또는 HYCMS 화면 구조가 바뀌면 동작이 깨질 수 있습니다.
- 비디오가 아닌 항목, 학습 예정 강의, 기간 종료 강의, 일부 대면/녹화 강의는 자동으로 스킵될 수 있습니다.
- 사용자의 네트워크 상태나 LMS 반영 지연에 따라 완료 확인까지 시간이 걸릴 수 있습니다.
- Chrome Web Store 배포를 목표로 한다면 `debugger` 권한 검토가 추가로 필요할 수 있습니다.

## 문제 해결

### 자동화가 시작되지 않을 때

- 현재 탭이 `learning.hanyang.ac.kr`인지 확인합니다.
- LMS 로그인 상태가 유지되는지 확인합니다.
- 확장 프로그램이 정상적으로 로드되었는지 `chrome://extensions`에서 확인합니다.

### 강의 진행이 멈춘 것처럼 보일 때

- 팝업의 상태 영역과 디버그 영역을 확인합니다.
- `페이지 디버그 패널 표시`를 켜고 어느 단계에서 멈추는지 확인합니다.
- LMS 탭을 새로고침한 뒤 다시 시작합니다.

### 완료 처리가 늦을 때

- `학습 상태 확인` 반영에는 약 5분 지연이 있을 수 있습니다.
- 필요하면 잠시 기다린 뒤 다시 확인합니다.
- 문제가 반복되면 `기존 완료 기록 초기화 후 시작`을 켠 뒤 다시 시도합니다.



### 버그 제보
